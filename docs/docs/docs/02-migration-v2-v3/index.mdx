# Migrating from 2.x to 3.x

## Environment Requirements

Cached Query 3.0 requires Dart 3.0.0 or higher. Make sure to update your `pubspec.yaml`:

```yaml
environment:
  sdk: ">=3.0.0 <4.0.0"
```
